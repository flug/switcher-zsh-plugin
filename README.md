Switcher persona plugin zsh
===========================


#Prerequis 

Install persona in you pc 

create many profiles

##Clone plugin :

```
    git clone https://github.com/flug/switcher-zsh-plugin.git ~/.oh-my-zsh/custom/plugins/switcher
```

##Add in you ```.zshrc``` 

```
    plugins=(switcher)
```
